jQuery Stepy is a plugin based on FormToWizard that generates a customizable wizard.
With default options:
description one
description two

$('#default').stepy();
A custom form in a validation style:
my description one
my description two

- When the attribute 'validate' is true, you must include the jquery.validate.js.

$('#custom').stepy({
  backLabel:      'Backward',
  block:          true,
  errorImage:     true,
  nextLabel:      'Forward',
  titleClick:     true,
  validate:       true
});

With transitions callback:
description one

$('#callback').stepy({
  onBack: function(index) {
    alert('Going to step ' + index + '...');
  },
  onNext: function(index) {
    if (!isValid()) {
      alert('Invalid value!');
      return false;
    }

    alert('Going to step ' + index + '...');
  },
  titleClick: true
});

- You can do validations during the callback;
- Return "false" to prevent the transition;
- Return "true" or nothing to continue the transition.
HTML structure:
<form id="custom">
  <fieldset title="Title">
    <legend>description</legend>
    <!-- input fields -->
  </fieldset>

  <fieldset title="Title">
    <legend>description</legend>
    <!-- input fields -->
  </fieldset>

  <input type="submit" class="finish"/>
</form>
The fieldset's title is the main title of the step and the legend is the description of it.
Default options:
backLabel:      '< Back'
Change the back button label.
block:          false
* Block the next step if the current is invalid.
errorImage:     false
* If an error occurs, a image is showed in the title of the corresponding step.
finish:         true
Include the button with class called '.finish' into the last step.
nextLabel:      'Next >'
Change the next button label.
onBack:         function(index) { return true; }
Callback before the backward action.
onNext:         function(index) { return true; }
Callback before the forward action.
titleClick:     true
Active the back and next action in the titles.
validate:       false
* Active the jQuery Validation for each step.
* Depends of jquery.validation.js.

- All public functions have a optional second parameter to specify which container will be executed;
- You can pass a ID to be the target of the action.
- If the ID is not specified, then the last element Stepy will be takes.
Public functions:
$.fn.stepy.step(2, '#stepy');
Change the form to step 2. Validation is not supported yet.
